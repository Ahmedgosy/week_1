# Week 1 - Flutter Task

This project is a simple Flutter UI built for the week 1 assignment.  
It includes:

- Header section
- Search bar
- Categories chips
- Top Rated Designers section
- Designers grid with cards

## How to run
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run` on any emulator or physical device
